HTML & CSS

La création des pages web nécessite l'usage d'un langage de description (le HTML ), et d'un langage de mise en forme (le CSS). Ces deux langages, avec le Java Script, constituent ainsi les bases du développement web. En ce moment, le développeur web est l'un des métiers les mieux rémunérés et de plus en plus de professionnels décident de se former aux technologies HTML et CSS afin de pouvoir démarrer une activité en freelance et trouver aisément des missions.

Définition et rôle du HTML

Le HTML (HyperText Markup Language)est un langage informatique utilisé pour la création de sites web. Dans le détail, ce langage sert à mettre en forme les pages web via une structure de balisage :

                             
               ●  insérer un titre, des sous-titres, 
                                                                            
               ●  mettre des mots ou des phrases en gras, 
                                                        
               ●  insérer du texte, des liens, des images ou des vidéos, etc.

Les balises vont toujours par paire, le nom de l'élément et l'attribut se trouve dans la balise d'ouverture, marquée par deux chevrons (˂nom de l'élément et attribut ˃), tandis que la balise de fermeture ne fait que reprendre le nom de l'élément précédé par un slash (˂nom de l'élément˃).

Une page codée en HTML ne peut être visualisée qu'avec un navigateur web. Ce langage doit être associé au langage CSS utilisé pour la mise en forme du contenu de la page web, et éventuellement avec JavaScript, pour ajouter plus d'interactivité.

la langage CSS

le CSS (Cascading style sheet ou feuille de style en cascade en français) est un langage de programmation utilisé pour la mise en forme d'une page web codée en HTML. Il s'agit ainsi d'un langage descriptif qui se complète aux langages de balisage ( HTML, XML). Le CSS comporte des règles de mise en forme pouvant être appliquées simultanément sur différents éléments ou documents. L'objectif est d'apporter un style visuel et d'améliorer l'esthétique d'un site web.

Pourquoi utiliser le HTML

La création d'un site web nécessite la production d'un contenu fixe en premier lieu, et pour cela, il faut utiliser le langage HTML . Cette étape permet d'obtenir des pages statiques pouvant être traduites visuellement par le navigateur web. Ensuite, il faut donner du style aux éléments des pages web à l'aide du langage CSS.

La maîtrise de ces deux langages informatiques est ainsi de rigueur pour celui qui envisage de créer un site web. Reconnus comme étant des langages simples, le HTML et le CSS sont bien accessibles aux débutants.
